1. complile the tool by 
    cc -o GenFiles GenFiles.c
2. run the tool by
   ./GenFiles SM_Info.txt
   ./Mfds.sh (you may need to modify the shell)
   ./DoA SM_Info.txt
   ./GSD  (this command is optional)
   ./FirePM SM_Info.txt
